Routenplaner hinzufügen:

1. route.html und route.js in dein GitHub-Repository hochladen.
2. In index.html oben oder irgendwo im Header diesen Link ergänzen:
   <a href="route.html">Routenplaner</a>
3. Commit changes klicken.
4. 1-2 Minuten warten.
5. Webseite mit Strg+F5 neu laden.
6. Routenplaner öffnen: https://ginaborgmann.github.io/vertriebler-karte/route.html

Hinweis: Kilometer und Minuten sind geschätzt per Luftlinie, nicht echte Straßenroute.
